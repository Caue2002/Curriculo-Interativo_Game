### Currículo Interativo - Cauê Ribeiro Ferreira

# Descrição do Projeto
O Currículo Interativo é uma aplicação web desenvolvida para apresentar o currículo de forma dinâmica e envolvente, combinando elementos de jogos 2D com informações profissionais. O usuário pode movimentar um personagem pelo mapa, explorar diferentes tópicos e visualizar dados como formação acadêmica, experiência profissional, habilidades, redes sociais e carta de apresentação.
O projeto visa demonstrar habilidades em HTML, CSS e JavaScript, além de criatividade na apresentação de informações de maneira interativa.


# Funcionalidades

Movimento do personagem usando teclado (W A S D ou setas).

Tópicos interativos que mostram popups com informações do currículo.

Imagem de comemoração aparece ao lado do popup ao acessar um tópico.

Popups exibem:

Dados pessoais e objetivo profissional.

Formação acadêmica e conhecimentos técnicos.

Experiência profissional e habilidades.

Redes sociais com links clicáveis.

Carta de apresentação.

Design responsivo para uso em diferentes tamanhos de tela.

Animação suave no personagem e efeitos visuais nos tópicos e popups.

Tecnologias Utilizadas

HTML5 – estrutura da página e elementos interativos.

CSS3 – estilos, layout, animações.

JavaScript (Vanilla JS) – lógica de movimento, colisões, exibição de popups e interatividade.


# Como Usar

Abra o arquivo index.html em um navegador moderno (Chrome, Edge, Firefox).

Movimente o personagem pelo mapa:

PC: use as teclas W A S D ou as setas do teclado.

Aproxime-se de um tópico para abrir o popup de informações.

Observe a imagem de comemoração aparecer ao lado do popup quando ele abrir.

Explore todos os tópicos para visualizar todas as informações do currículo.
